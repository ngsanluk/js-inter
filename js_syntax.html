JavaScript Syntax

Variable and Data Type

  • variable names can contain letters, digits, underscores, and dollar signs.
  • variable names must begin with a letter
  • variable names are case sensitive (y and Y are different variables)
  • JavaScript data types include string, number, boolean, array and object
  • you can use typeof keyword to check a type of a variable
  • JavaScript is dynamic types, meaning the same variable can be used to hold different data types

Arithmetic Operators

+ : Addition
- : Subtraction
* : Multiplication
** : Exponentiation (ES2016)
/ : Division
% : Modulus (Division Remainder)
++ : Increment
-- : Decrement 

 x + y;
 x - y;
 x * y;
 x / y;      
 x % y;
 x++;
 y--;

Comparison Operators

!= not equal

> greater than
< less than
